# AutomationEDA (UAS EDA)

Repository ini berisi source code **aplikasi dashboard** untuk kebutuhan **tugas UAS EDA**.

## Deskripsi Singkat
Aplikasi dibuat menggunakan **Next.js (App Router)**, **TypeScript**, dan komponen **Shadcn UI**.

## Cara Menjalankan (Local)
1. Install dependensi:
   ```bash
   npm install
   ```
2. Jalankan development server:
   ```bash
   npm run dev
   ```
3. Buka:
   - http://localhost:3000

## Build
```bash
npm run build
npm start
```

## Catatan
Silakan sesuaikan konfigurasi (.env) bila diperlukan pada lingkungan masing-masing.
